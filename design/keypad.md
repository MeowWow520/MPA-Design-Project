# KEYPAD-PHONE 接线说明

## 一、硬件连接（基于当前代码）

```
80C51                          KEYPAD-PHONE (4×3)
┌──────────┐                   ┌──────────────┐
│ P3.0 ────┼───────────────────┤ R1 (Row 0)   │
│ P3.1 ────┼───────────────────┤ R2 (Row 1)   │
│ P3.2 ────┼───────────────────┤ R3 (Row 2)   │
│ P3.3 ────┼───────────────────┤ R4 (Row 3)   │
│          │                   │              │
│ P3.4 ◄───┼───────────────────┤ C1 (Col 0)   │
│ P3.5 ◄───┼───────────────────┤ C2 (Col 1)   │
│ P3.6 ◄───┼───────────────────┤ C3 (Col 2)   │
└──────────┘                   └──────────────┘
```

| MCU 引脚 | 方向 | KEYPAD 引脚 | 说明 |
|----------|------|-------------|------|
| P3.0 | 输出 | R1 (Row 0) | 扫描时拉低 |
| P3.1 | 输出 | R2 (Row 1) | 扫描时拉低 |
| P3.2 | 输出 | R3 (Row 2) | 扫描时拉低 |
| P3.3 | 输出 | R4 (Row 3) | 扫描时拉低 |
| P3.4 | 输入 | C1 (Col 0) | 读列状态 |
| P3.5 | 输入 | C2 (Col 1) | 读列状态 |
| P3.6 | 输入 | C3 (Col 2) | 读列状态 |

> 80C51 P3 口自带内部上拉电阻，列线无需外接电阻。扫描时将当前行拉低，读列线：低电平 = 按键按下。

## 二、按键矩阵与键值映射

```
        C1(P3.4)  C2(P3.5)  C3(P3.6)
          │         │         │
R1(P3.0)─1─────────2─────────3──
R2(P3.1)─4─────────5─────────6──
R3(P3.2)─7─────────8─────────9──
R4(P3.3)─*─────────0─────────#──
```

| 按键 | 行 | 列 | 键值宏 | 数值 | 游戏功能 |
|------|-----|-----|--------|------|---------|
| `2` | R1 | C2 | `KEY_UP` | 2 | 玩家向上移动 |
| `8` | R3 | C2 | `KEY_DOWN` | 8 | 玩家向下移动 |
| `#` | R4 | C3 | `KEY_START` | 12 | 开始 / 暂停 / 重新开始 |

## 三、扫描时序

```
for row = 0 to 3:
    拉低 R(row)               // ROWx = 0
    延迟 1ms                  // 等待电平稳定
    读 C1, C2, C3             // 哪列为低即哪个键按下
    拉高 R(row)               // ROWx = 1
```

一行扫描完毕立即恢复高电平，再扫下一行。完整扫描 4 行约需 4ms。

## 四、软件消抖

```
prev ← 上次读取值
cnt  ← 计数器

每帧:
  raw ← 当前扫描键值
  if raw == prev AND raw != KEY_NONE:
    cnt++
    if cnt >= 2:
      cnt ← 0
      return raw        // 确认按下
  else:
    cnt ← 0
  prev ← raw
  return KEY_NONE
```

连续 2 帧读到相同非空值才确认按键，约 600ms 消抖时间（300ms/帧）。

## 五、Proteus 操作要点

- 元件名：`KEYPAD-PHONE`（在 Proteus 元件库中搜索）
- 引脚排列通常为上下两排，需对照元件属性确认 `R1~R4`、`C1~C3` 的具体位置
- 无需外接上拉电阻（80C51 P3 内部上拉）
- 接线无误后，可用电压探针验证：按下按键时对应行列线应均为低电平
